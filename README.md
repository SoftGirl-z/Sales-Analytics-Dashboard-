# Sales Analytics Dashboard — Power BI

A business intelligence dashboard analyzing Q4 2024 sales performance
across product categories and geographic regions.

## Key Metrics Tracked
- Total Revenue: $4.82M (+18.4% QoQ)
- Gross Margin: 61.3%
- Churn Rate: 3.7%

## Visuals & Features
- Monthly revenue vs. target (grouped bar chart)
- Category breakdown via donut chart
- 12-month customer growth trend
- Regional performance ranking

## Tools Used
- Microsoft Power BI Desktop
- DAX (calculated measures, YoY comparisons)
- Power Query (data transformation)
- Sample dataset: Contoso Sales Dataset

## How to Use
1. Download `SalesDashboard.pbix`
2. Open in Power BI Desktop
3. Refresh data or connect your own source
